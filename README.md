# Node React Learning Project

A simple React project exploring hooks and props.

## Files
- `hooks.jsx`: Demonstrates `useState` and state management.
- `src/math.jsx`: Demonstrates passing props to child components.
- `src/User.jsx`: A reusable UI component for displaying user data.
- `ca.js`: Arduino-style hardware control code.

## How to run
This project is currently a set of learning components. To run them, you can integrate them into a Vite or Create React App environment.
